Git:- Git is version control system which allows us to store project history, i.e. which changes made by whom in project

GitHub:- Git hub is online platform to provide hosting our project, repositories online so that other people can see and share their
contribution, other platforms are bitbucket, Gitlab, git bucket

example: Team of five is working on project then how they can track each other code and share their changes in project can be
done by git and git hub
